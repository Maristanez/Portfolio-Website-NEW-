# 3D Developer Portfolio

Interactive 3D portfolio website built with modern web tech to showcase my projects, skills, and experience.

## Tech Stack

- React (or Next.js) for the frontend
- Three.js / React Three Fiber for 3D scenes
- TypeScript for type-safe development
- Vite / Next.js dev server and bundler
- CSS / Tailwind CSS for styling

## Features

- Hero section with 3D model and intro
- Projects grid with live links and source code
- Skills section with icons / visualizations
- About section with background and interests
- Contact form or links to email and socials
